This is a [Next.js](https://nextjs.org) project bootstrapped manually in this environment.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Included setup

- Next.js (latest)
- React / React DOM (latest)
- TypeScript
- ESLint (Next config)
- Tailwind CSS
- App Router (`src/app`)
