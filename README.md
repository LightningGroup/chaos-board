# Chaos Board

찬반 투표와 실시간 토론을 하나의 화면에서 해결하고 싶어서 만든 페이지.

## Screenshots

![Main Screen](./chaos-board.png)

## Quick Start

```bash
# 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프로덕션 실행
npm start
```

`http://localhost:3000` 에서 확인 가능.

## Tech Stack

- Next.js (App Router)
- React 18
- TypeScript
- Tailwind CSS
